## CineCloud – AI-Powered Movie Intelligence Platform

CineCloud is a PHP-based MVC web application that allows users to search movies, rate them, and manage reviews in a structured architecture. The application follows the Model-View-Controller (MVC) design pattern and demonstrates clean separation of concerns, routing, session management, and database integration.

## Architecture

The project follows a custom MVC structure:

cinecloud/
│
├── app/
│   ├── controllers/
│   ├── models/
│   ├── views/
│   └── core/
│
├── public/
│   ├── index.php
│   └── .htaccess
│
└── README.md

## How to Run on Replit
This project runs directly on Replit — no XAMPP or local setup required.

Steps:
1. Copy the GitHub Clone URL of this repository
2. Go to Replit
3. Click Import Code
4. Select “Import from GitHub”
5. Paste the clone URL
6. Click Import

Application will run instantly in the browser preview.
